# GenerateChart MCP Server

## 数据格式

### 柱状图 / 条形图 / 堆叠条形图 / 折线图

```json
{
  "xLabels": "[\"一月\",\"二月\",\"三月\"]",
  "seriesData": "[{\"name\":\"销量\",\"values\":[120,200,150]}]"
}
```

### 饼图

```json
{
  "labels": "[\"苹果\",\"香蕉\",\"橘子\"]",
  "values": "[30,20,50]"
}
```

### 气泡图

```json
{
  "seriesData": "[{\"name\":\"城市\",\"x\":[1,2,3],\"y\":[10,20,30],\"size\":[40,50,60]}]"
}
```
